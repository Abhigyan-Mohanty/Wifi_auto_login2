As most of you know, it's quite annoying to manually inpu tthe user and password of your own or "your friend's" into the annoying website which pops up everytime you reconnect your computer to the internet. So I built this tool where you can input all users and passwords you have into /config/credentials in the format required, and it automatically inputs the details. Further if the data limit on one of the account runs out, it cycles onto the next account. Have fun with this!
